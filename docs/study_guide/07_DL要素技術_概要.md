# 07 DL要素技術 ── 概要・重要用語集

---

## 📖 章の概要

深層学習の主要な発展技術を扱う。Attention・Transformer・BERT・GPT、転移学習、強化学習の発展「深層強化学習」など。

---

## 🗂️ 大項目一覧と要点

### Attention機構とTransformer
| 用語 | 内容 |
|---|---|
| **Attention機構** | 系列の各要素が他の要素を参照する重み付き注意機構 |
| **Self-Attention** | 系列内の各要素が互いに関係性を計算 |
| **Multi-Head Attention** | 複数の注意ヘッドで異なる表現を同時に学習 |
| **Transformer** | Attentionのみに基づいた並列処理可能なモデル（2017, Google「Attention is All You Need」） |
| **エンコーダ** | 入力系列をベクトル表現に変換する部分 |
| **デコーダ** | ベクトル表現から出力系列を生成する部分 |
| **位置エンコーディング** | Transformerに系列の位置情報を与えるための工夫 |

### BERTとGPT系モデル
| モデル | 特徴 |
|---|---|
| **BERT** | 双方向Transformer。マスク言語モデリング（MLM）と次文予測（NSP）で事前学習（Google, 2018） |
| **GPT** | 単方向Transformer（左→右）。次トークン予測で事前学習（OpenAI） |
| **GPT-3** | 1750億パラメータ。フューショット学習が可能 |
| **ChatGPT** | GPTをRLHFでファインチューニングした対話特化モデル |
| **RLHF** | 人間のフィードバックによる強化学習。ChatGPT等に使用 |

### 転移学習・ファインチューニング
| 用語 | 内容 |
|---|---|
| **転移学習** | 別タスクで事前学習したモデルを新タスクに適用 |
| **ファインチューニング** | 事前学習済みモデルのパラメータを新タスク向けに追加学習 |
| **プロンプトチューニング** | モデルのパラメータを固定し、プロンプトのみを最適化 |
| **LoRA** | 低ランク行列で分解して効率的にファインチューニング |
| **RAG（検索拡張生成）** | 外部データベースから関連情報を検索してLLMに渡す手法 |

### 深層強化学習
| 用語 | 内容 |
|---|---|
| **DQN（Deep Q-Network）** | ニューラルネットワークでQ値を学習。DeepMindがAtariゲームで成果 |
| **Q学習** | 行動の価値（Q値）を学習する強化学習の基本アルゴリズム |
| **方策勾配法** | 方策（行動決定ルール）を直接を最適化 |
| **Actor-Critic** | 方策（Actor）と価値（Critic）を同時に学習 |
| **AlphaGo** | 囲碁AI。深層強化学習+モンテカルロ木探索（DeepMind, 2016） |
| **AlphaZero** | 人間の棋譜なしで将棋・チェス・囲碁をマスター |

### 生成モデル
| モデル | 特徴 |
|---|---|
| **オートエンコーダ（AE）** | 入力を圧縮（エンコード）→再構成（デコード）して特徴を学習 |
| **VAE（変分オートエンコーダ）** | 確率分布から潜在変数をサンプリングして多様なデータ生成 |
| **GAN（敵対的生成ネットワーク）** | Generator vs Discriminatorの対立的学習 |
| **拡散モデル** | 拡散過程（ノイズ付与）→逆拡散過程（ノイズ除去）で画像生成 |

### 主要NLPの前処理・単語表現
| 用語 | 内容 |
|---|---|
| **トークン化** | テキストを単語・サブワードに分割 |
| **埋め込み（Embedding）** | 単語をベクトルに変換（意味を保持） |
| **Word2Vec** | 近接単語の共起から単語ベクトルを学習 |
| **CBOW** | 周辺語から中心語を予測（Word2Vecの手法） |
| **Skip-gram** | 中心語から周辺語を予測（Word2Vecの手法） |
