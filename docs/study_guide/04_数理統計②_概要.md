# 04 数理統計② ── 概要・重要用語集

> **G検定 クイックリファレンス** | 分野：数理統計② | 2026-02-25

---

## 📖 章の概要

統計的推定・検定、最適化手法（勾配降下法）、グラフ理論・ベイズ推定を扱う章。機械学習モデルの学習の仕組みを数学的に理解する。

---

## 🗂️ 大項目一覧と要点

### 統計的推定
| 用語 | 内容 |
|---|---|
| **点推定** | 標本から母数を1点で推定 |
| **区間推定** | 信頼区間で母数の範囲を推定 |
| **信頼区間** | 「95%信頼区間」= 95%の確率で母数を含む区間 |
| **最尤推定** | 観測データが最も生じやすくなるパラメータを推定 |
| **MAP推定** | 事前分布を考慮した最尤推定（最大事後確率推定） |

### 仮説検定
| 用語 | 内容 |
|---|---|
| **帰無仮説 (H₀)** | 否定したい仮説（効果なし・差がない） |
| **対立仮説 (H₁)** | 採択したい仮説（効果あり・差がある） |
| **p値** | 帰無仮説が正しい場合に観測値以上の結果が出る確率 |
| **有意水準** | 帰無仮説を棄却するp値の閾値（通常5%や1%） |
| **第1種の誤り** | 帰無仮説が正しいのに棄却する（偽陽性） |
| **第2種の誤り** | 帰無仮説が誤りなのに採択する（偽陰性） |
| **多重検定問題** | 検定を繰り返すことで偽陽性率が増加する問題 |

### 最適化手法
| 用語 | 内容 |
|---|---|
| **損失関数（誤差関数）** | 予測と正解のズレを数値化する関数 |
| **勾配降下法** | 損失関数の勾配（傾き）に沿ってパラメータを更新 |
| **学習率（ステップサイズ）** | 1回の更新で移動する量。大きすぎると発散、小さすぎると遅い |
| **SGD（確率的勾配降下法）** | ミニバッチを使った効率的な勾配降下法 |
| **局所最適解** | 損失関数の局所的な最小値（大域最適解ではない可能性あり） |
| **大域最適解** | 損失関数全体の最小値 |
| **鞍点** | 勾配がゼロだが最小でも最大でもない点 |

### 最適化発展手法
| 手法 | 特徴 |
|---|---|
| **Momentum** | 過去の勾配を蓄積して慣性的に更新 |
| **AdaGrad** | パラメータごとに学習率を調整（頻度が高い特徴は小さく） |
| **RMSprop** | AdaGradの学習率減衰を改善 |
| **Adam** | MomentumとRMSpropを組み合わせた手法。現在最も広く利用 |

### ベイズ推定
| 用語 | 内容 |
|---|---|
| **ベイズ推定** | 事前分布を観測データで更新して事後分布を求める |
| **共役事前分布** | 事後分布が事前分布と同じ分布族になる都合の良い事前分布 |
| **MCMC（マルコフ連鎖モンテカルロ法）** | 事後分布からサンプリングする手法 |

### グラフ理論
| 用語 | 内容 |
|---|---|
| **グラフ（G）** | ノード（頂点）とエッジ（辺）からなるデータ構造 |
| **有向グラフ** | エッジに方向がある（→） |
| **無向グラフ** | エッジに方向がない |
| **DAG（有向非循環グラフ）** | 循環のない有向グラフ。ベイズネットワーク等に使用 |
