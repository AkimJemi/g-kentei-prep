## 🗂️ 大項目一覧と要点

### 人工知能の定義
- 「人工知能」の命名：**ジョン・マッカーシー**（ダートマス会議、1956年）
- 定義は研究者によって**異なる**（松尾豊、浅田稔 等）

### AIの分類（レベル別）
| レベル | 内容 | 例 |
|---|---|---|
| 1 | ルールベース制御 | 冷蔵庫・炊飯器 |
| 2 | 古典的AI | エキスパートシステム |
| 3 | 機械学習 | スパムフィルタ |
| 4 | ディープラーニング | 画像認識・音声認識 |

### AI研究の歴史
| ブーム | 時代 | キーワード |
|---|---|---|
| 第1次 | 1950s〜60s | 推論・探索・トイプロブレム・**組み合わせ爆発** |
| 第2次 | 1980s | **エキスパートシステム**・知識獲得ボトルネック |
| 第3次 | 2000s〜 | 機械学習・**ディープラーニング**・特徴表現学習 |
| 第4次 | 2020s〜 | **生成AI**・LLM・ChatGPT |

### 主要な問題と課題
| 用語 | 内容 |
|---|---|
| **トイプロブレム** | ルールが明確な単純問題。現実問題へ非対応 |
| **フレーム問題** | 関連事象と無関係事象を区別できない（提唱：マッカーシー＋ヘイズ） |
| **チューリングテスト** | AIが人間と区別されないか検証（**アラン・チューリング**、1950年） |
| **中国語の部屋** | 記号操作≠理解（**ジョン・サール**、1980年） |
| **AI効果** | AI技術が実用化されると「これはAIではない」と見なされる心理 |

### 機械翻訳の進化
1. **ルールベース** → 人間がルール作成、精度低
2. **統計的機械翻訳** → コーパスから学習（Google翻訳初期）
3. **ニューラル機械翻訳** → NMT・GNMT（2016年Google）、精度飛躍的向上

### シンギュラリティ
- 提唱者：**ヴァーナー・ヴィンジ**
- **レイ・カーツワイル**の予測：2029年AI > 人類、**2045年**シンギュラリティ到来
- 脅威と見る人：**イーロン・マスク**・**ビル・ゲイツ**・**スティーブン・ホーキング**

### 主要国際会議
| 略称 | 分野 |
|---|---|
| CVPR | コンピュータビジョン・パターン認識 |
| NeurIPS | ニューラルネットワーク・情報処理 |
| ICML | 機械学習全般 |
| IJCAI | AI全般（AAAIが主催） |
| AAAI | AI全般（1979年設立の非営利団体） |
