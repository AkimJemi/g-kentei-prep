# 11 法律契約 ── 概要・重要用語集

> **G検定 クイックリファレンス** | 分野：法律契約 | 2026-02-25

---

## 📖 章の概要

AI関連の知的財産権（著作権・特許権）、個人情報保護法（GDPR含む）、AIと契約、AI生成物の権利問題を扱う章。

---

## 🗂️ 大項目一覧と要点

### 知的財産権の基礎
| 権利 | 内容 | 期間 |
|---|---|---|
| **著作権** | 創作表現物（文章・絵・音楽等）を保護。登録不要・自動発生 | 著作者の死後70年 |
| **特許権** | 新規かつ有用な発明を保護。登録が必要 | 出願から20年 |
| **商標権** | 商品・サービスに使用するロゴ・名称等を保護 | 10年（更新可） |
| **意匠権** | 製品の形状・模様・色彩等のデザインを保護 | 登録から25年 |
| **不正競争防止法** | 営業秘密・顧客情報等の不正取得・使用を禁止 | — |

### 著作権の重要事項
| 用語 | 内容 |
|---|---|
| **著作物** | 思想・感情を「創作的に」表現したもの（アイデア自体は保護対象外） |
| **著作者** | 著作物を創作した人 |
| **著作権者** | 著作権を有する人（著作者本人または譲受人） |
| **著作財産権** | 複製権・翻訳権・公衆送信権等の財産的権利。第三者への譲渡可 |
| **著作人格権** | 公表権・氏名表示権・同一性保持権。**譲渡不可** |
| **法人著作（職務著作）** | 法人の発意で従業員が作成した著作物は法人が著作者となる |
| **引用** | 出所を明示し、主従関係が明確であれば他者著作物を利用可能 |

### AI生成物と著作権
| 状況 | 著作権の扱い |
|---|---|
| **人間がAIを道具として使用** | 人間の創作的判断が加わる場合、人間に著作権 |
| **AIが自律的に生成** | 日本では現時点で著作権は発生しない（創作主体が人ではないため） |
| **AIのプロンプト設計** | 高度な創作的関与があれば著作権が認められる可能性あり |
| **学習データの著作権** | 情報解析目的の学習は著作権法30条の4で原則許容（日本） |

### 個人情報保護法
| 用語 | 内容 |
|---|---|
| **個人情報** | 氏名・生年月日等により特定の個人を識別できる情報 |
| **要配慮個人情報** | 人種・信条・病歴・犯罪歴等の特に慎重な取扱いが必要な情報 |
| **匿名加工情報** | 特定の個人を識別できないよう加工した情報。第三者提供可 |
| **仮名加工情報** | 他情報と照合しなければ識別できないよう加工した情報 |
| **個人情報保護委員会** | 個人情報保護法の監督機関（日本） |
| **3年ごと見直し原則** | 個人情報保護法はデジタル環境変化に対応するため頻繁に改正 |

### GDPR（EU一般データ保護規則）
| 用語 | 内容 |
|---|---|
| **GDPR** | EU/EEA域内の個人データ保護の包括的法規制（2018年施行） |
| **適用範囲** | EU/EEA内の個人データを扱う全組織（EU外企業も対象） |
| **主な権利** | 知る権利・アクセス権・訂正権・削除権（忘れられる権利）・ポータビリティ権 |
| **忘れられる権利** | 個人がデータ消去を要求できる権利 |
| **データポータビリティ権** | 自分のデータを別のサービスに移行できる権利 |
| **DPO（データ保護責任者）** | 大規模データ処理企業に義務付けられる役割 |
| **違反時の罰則** | 最大2000万ユーロまたは全世界売上4%のどちらか高い額 |

### AIと契約
| 問題 | 内容 |
|---|---|
| **責任の帰属** | AI判断による事故・損害は開発者・利用者のどちらが負うか |
| **製造物責任（PL法）** | AIを組み込んだ製品の欠陥による損害の責任 |
| **プロバイダ責任制限法** | プラットフォーム事業者の第三者コンテンツへの責任を限定 |

### 営業秘密と不正競争防止法
| 用語 | 内容 |
|---|---|
| **営業秘密** | 秘密管理性・有用性・非公知性の3要件を満たすもの |
| **限定提供データ** | 特定の者に有償で提供されるデータ（AI学習用データ等） |
| **不正競争行為** | 他社の営業秘密を不正に取得・使用する行為 |
