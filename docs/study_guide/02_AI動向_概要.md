## 🗂️ 大項目一覧と要点

### AI研究の3時代
| 時代 | キーワード |
|---|---|
| 推論・探索 | 探索木、幅優先・深さ優先、Mini-Max法 |
| 知識 | エキスパートシステム、SHRDLU、Cycプロジェクト |
| 機械学習・特徴表現学習 | ディープラーニング、特徴量、ビッグデータ |

### 探索アルゴリズム
| アルゴリズム | 特徴 |
|---|---|
| **幅優先探索（BFS）** | 最短経路を保証、メモリ消費大 |
| **深さ優先探索（DFS）** | メモリ消費小、最短経路は非保証 |
| **Mini-Max法** | ボードゲームで有効。自分: Max、相手: Min |
| **αβ法** | Mini-Max改良版。αカット・βカットで計算量削減 |

### 知識表現
| 概念 | 内容 |
|---|---|
| **意味ネットワーク** | ノード（概念）+ リンク（関係）で知識を表現 |
| **is-a** | 継承関係（推移律が成立） |
| **part-of** | 部分関係（推移律は常に成立しない） |
| **has-a** | 所有関係 |

### オントロジー
| 種類 | 特徴 | 代表例 |
|---|---|---|
| **ヘビーウェイト** | 厳密性・コスト大 | Cycプロジェクト |
| **ライトウェイト** | 効率性重視 | ワトソン（IBM、**拡張知能**） |

- **オントロジーの定義**（トム・グルーパー）：「概念化の明示的な仕様」
- **セマンティック・ウェブ**：Webページにメタデータを付与してコンピュータが自律処理
- **LOD（リンクト・オープン・データ）**：構造化データの公開・共有技術

### 機械学習
- 定義（**アーサー・サミュエル**）：「明示的にプログラムしなくても学習する能力を与える研究分野」
- **パターン認識**：画像・音声・テキストからパターンを識別
- **みにくいアヒルの子定理**：分類するには仮定に基づく特徴量選択が必要
- **統計的自然言語処理**：コーパスを活用した翻訳・NLP

### ディープラーニング
- **ILSVRC 2012**：AlexNetが優勝、**ジェフリー・ヒントン**主導
- **特徴表現学習**：モデルが自動的に特徴量を抽出
- **2015年**：ディープラーニングの認識精度が人間を超える

### 大規模言語モデル（LLM）
| 用語 | 内容 |
|---|---|
| **LLM** | 膨大テキストで学習、自然言語タスクをこなすモデル |
| **ChatGPT** | サービス名（GPTをベースとした対話AI） |
| **GPT** | モデル名（Generative Pre-trained Transformer） |
| **Transformer** | LLMの核となるニューラルネットワーク構造（Google） |
| **事前学習** | 大量データで汎用的な知識を学習 |
| **ファインチューニング** | 特定タスク向けに追加学習 |
| **生成AI** | テキスト・画像・音声などを新たに生成するAI |
