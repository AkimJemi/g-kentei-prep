## 01 AI概論

| 用語 | 内容 |
|---|---|
| **人工知能（AI）** | 人間の知的行動をコンピュータで実現する技術の総称 |
| **汎用AI（AGI）** | あらゆるタスクに対応できる人間レベルのAI（未実現） |
| **特化型AI（ANI）** | 特定タスクに限定したAI（現在の主流） |
| **AIの冬** | 期待と現実のギャップから研究が停滞した時期（2度） |
| **第3次AIブーム** | 機械学習・深層学習の台頭（2010年代〜） |
| **チューリングテスト** | 人間がAIと人間を区別できなければ合格とするテスト |
| **シンギュラリティ** | AIが人間の知能を超える技術的特異点（カーツワイル） |
| **フレーム問題** | AIが「今考慮すべきこと以外」を無視できない問題 |
| **記号接地問題** | AIが記号の意味を理解できないという本質的問題 |

⭐ `汎用AI vs 特化型AI` `AIの冬（2度）` `チューリングテスト` `シンギュラリティ` `フレーム問題` `記号接地問題`

## 03 数理統計①

| 用語 | 内容 |
|---|---|
| **平均・分散・標準偏差** | 分布の中心・ばらつきを表す基本統計量 |
| **正規分布** | 平均±1σに68%、±2σに95%、±3σに99.7%のデータ |
| **ベイズの定理** | P(A\|B) = P(B\|A)×P(A) / P(B)。事後確率の更新 |
| **最尤推定（MLE）** | データが得られる確率（尤度）を最大化するパラメータ推定 |
| **MAP推定** | 事前分布×尤度を最大化。ベイズ的最適化 |
| **中心極限定理** | サンプル平均は標本数が大きくなると正規分布に近づく |
| **相関係数** | -1〜1。因果関係とは別（相関≠因果） |
| **多変量解析** | 複数変数を同時に扱う統計手法 |
| **仮説検定** | 帰無仮説を棄却するかどうかの統計的判断（p値） |

⭐ `正規分布（68-95-99.7則）` `ベイズの定理` `最尤推定` `相関≠因果` `p値・有意水準`

## 04 数理統計②

| 用語 | 内容 |
|---|---|
| **勾配降下法** | 損失関数の勾配方向にパラメータを更新する最適化手法 |
| **確率的勾配降下法（SGD）** | ランダムに1サンプルで勾配計算。ミニバッチ版が一般的 |
| **Adam** | モーメンタム＋RMSPropを組み合わせた高性能最適化手法 |
| **学習率** | パラメータ更新の刻み幅（η）。大きすぎると発散 |
| **損失関数** | モデルの予測誤差を定量化する関数。最小化が目標 |
| **交差エントロピー** | 分類問題の損失関数。確率分布の差異を測定 |
| **二乗誤差（MSE）** | 回帰問題の損失関数 |
| **行列・ベクトル演算** | DNNの順伝播・逆伝播の基礎は線形代数 |
| **シグモイド微分** | σ(x)(1-σ(x))。勾配消失の原因 |

⭐ `勾配降下法` `SGD→Adam` `学習率` `交差エントロピー（分類）` `MSE（回帰）`

## 05 機械学習

| 種類 | 特徴 |
|---|---|
| **教師あり学習** | 正解ラベル付きデータで学習。分類・回帰問題に対応 |
| **教師なし学習** | 正解ラベルなし。クラスタリング・次元削減に使用 |
| **半教師あり学習** | 少量のラベル付き＋大量のラベルなしデータを活用 |
| **強化学習** | エージェントが報酬を最大化するよう試行錯誤 |

| アルゴリズム | 用途 | キーポイント |
|---|---|---|
| **SVM** | 分類 | マージン最大化、カーネル法で非線形対応 |
| **ランダムフォレスト** | 分類・回帰 | アンサンブル（バギング）、過学習に強い |
| **XGBoost** | 分類・回帰 | ブースティング代表。競技ML最強クラス |
| **k-means** | クラスタリング | k個のクラスタに分類（非階層型） |
| **PCA** | 次元削減 | 分散が最大となる方向に圧縮 |

| 評価指標 | 計算式 | 場面 |
|---|---|---|
| **適合率（Precision）** | TP/(TP+FP) | FP（偽陽性）を減らしたい |
| **再現率（Recall）** | TP/(TP+FN) | FN（偽陰性）を減らしたい（医療等） |
| **F値（F1）** | 調和平均 | バランスよく評価 |
| **AUC** | ROC曲線の面積 | 閾値によらないモデル比較 |

⭐ `教師あり・なし・強化学習` `SVM・マージン最大化` `バギング・ブースティング` `XGBoost` `混同行列` `適合率・再現率トレードオフ` `F値` `AUC` `L1（Lasso）・L2（Ridge）正則化` `AIC・BIC`

## 06 DL概要

| 用語 | 内容 |
|---|---|
| **多層パーセプトロン（MLP）** | 隠れ層を持つニューラルネットワーク |
| **ReLU** | 最も一般的な活性化関数。0以下は0、正値はそのまま |
| **ソフトマックス** | 多クラス分類の出力層。確率の総和=1 |
| **誤差逆伝播法** | チェーンルールで勾配を出力→入力方向に計算 |
| **勾配消失問題** | シグモイドの微分が0に近く、深い層まで勾配が届かない |
| **バッチ正規化** | 各バッチの活性値を正規化。学習安定化・高速化 |
| **ドロップアウト** | 学習時にランダムにニューロンを無効化。過学習防止 |
| **CNN** | 畳み込み層（フィルタ）＋プーリング層で画像特徴を抽出 |
| **LSTM** | ゲート機構で長期依存問題を解決したRNN |
| **GRU** | LSTMを簡略化した高効率版RNN |

⭐ `ReLU（現在主流）` `ソフトマックス` `誤差逆伝播法・チェーンルール` `勾配消失（シグモイド問題）` `バッチ正規化` `ドロップアウト` `CNN畳み込み・プーリング` `LSTM・GRU`

## 07 DL要素技術

| 用語 | 内容 |
|---|---|
| **Transformer** | Attentionのみに基づく並列処理可能なモデル（2017） |
| **Self-Attention** | 系列内の各要素が互いに関係性を計算 |
| **BERT** | 双方向Transformer。マスク言語モデリング（MLM）で事前学習 |
| **GPT** | 単方向Transformer（左→右）。次トークン予測で事前学習 |
| **RLHF** | 人間フィードバックによる強化学習。ChatGPT等に使用 |
| **RAG（検索拡張生成）** | 外部DBから関連情報を検索してLLMに渡す手法 |
| **LoRA** | 低ランク行列で効率的にファインチューニング |
| **DQN** | ニューラルネットでQ値を学習。DeepMindがAtariゲームで成果 |
| **AlphaGo** | 深層強化学習＋モンテカルロ木探索（DeepMind, 2016） |
| **GAN** | Generator vs Discriminatorの対立的学習 |
| **拡散モデル** | ノイズ付与→ノイズ除去の逆拡散で高品質生成 |
| **Word2Vec** | 近接単語の共起から単語ベクトルを学習（CBOW・Skip-gram） |

⭐ `Transformer（Attention is All You Need）` `BERT（双方向）` `GPT（単方向）` `RLHF` `RAG` `LoRA` `DQN・AlphaGo` `VAE・GAN・拡散モデル` `Word2Vec・CBOW・Skip-gram`

## 08 DL応用

| 用語 | 内容 |
|---|---|
| **AlexNet** | ILSVRC 2012優勝。DL普及の契機（2012） |
| **ResNet** | 残差接続（スキップ接続）で100層超えを実現（2015） |
| **YOLO** | 1パスで位置とクラスを同時予測。高速物体検出 |
| **CLIP** | 画像とテキストのペア学習。Zero Shot Learningが可能 |
| **拡散モデル** | 拡散（ノイズ付与）→逆拡散（ノイズ除去）で生成 |
| **Pix2Pix** | 条件付きGAN（CGAN）。ペア画像が必要 |
| **Cycle GAN** | ペア画像不要のCGAN。2ドメイン相互変換 |
| **基盤モデル** | 大規模事前学習した汎用モデル（ファインチューニングで特化） |
| **LIME / SHAP** | 説明可能AI（XAI）の手法。特徴量貢献度を可視化 |
| **蒸留（Distillation）** | 大きなモデルの知識を小さなモデルへ転移 |
| **量子化** | 重みのビット数削減（32bit→8bit等）。推論高速化 |
| **宝くじ仮説** | 大きなネットワーク内に小さな高性能サブネットが存在する |

⭐ `AlexNet・ResNet（残差接続）` `YOLO` `Pix2Pix（ペア必要）` `Cycle GAN（ペア不要）` `CLIP・Zero Shot` `基盤モデル` `LIME・SHAP・Grad-CAM` `蒸留・量子化・プルーニング` `宝くじ仮説`

## 11 法律契約

| 用語 | 内容 |
|---|---|
| **著作権** | 死後70年保護。登録不要・自動発生 |
| **著作人格権** | 公表権・氏名表示権・同一性保持権。**譲渡不可** |
| **著作財産権** | 複製権・公衆送信権等。**第三者への譲渡可** |
| **法人著作（職務著作）** | 法人の発意で従業員が作成→法人が著作者 |
| **AI生成物の著作権** | AIが自律的に生成→日本では現時点で著作権は発生しない |
| **著作権法30条の4** | 情報解析目的の学習は著作権侵害にならない（日本） |
| **個人情報** | 氏名・生年月日等により特定個人を識別できる情報 |
| **要配慮個人情報** | 人種・信条・病歴・犯罪歴等。特に慎重な取扱いが必要 |
| **匿名加工情報** | 識別できないよう加工した情報。第三者提供可 |
| **GDPR** | EU一般データ保護規則（2018年施行）。違反時は最大全世界売上4% |
| **忘れられる権利** | データ消去を個人が要求できる権利（GDPR） |
| **営業秘密（3要件）** | 秘密管理性・有用性・非公知性をすべて満たすもの |
| **特許権** | 出願から20年保護。登録が必要 |

⭐ `著作権（死後70年・自動発生）` `著作人格権（譲渡不可）` `AI学習データ（30条の4）` `AI生成物に著作権なし（原則）` `個人情報・要配慮個人情報` `GDPR（罰則4%）` `忘れられる権利` `営業秘密（3要件）`
