## 03 数理統計①

### 線形代数の基礎
| 用語 | 内容 |
|---|---|
| **スカラー / ベクトル / 行列** | 数値の次元の違い |
| **転置行列** | 行と列を入れ替えた行列（$A^T$）|
| **内積** | ベクトル同士の積のスカラー。類似度計算に使用 |
| **逆行列** | $A \cdot A^{-1} = I$ となる行列 |
| **固有値・固有ベクトル** | $Ax = \lambda x$ を満たす $\lambda$（固有値）と $x$（固有ベクトル），PCAの基礎 |

### 確率の基礎
| 用語 | 内容 |
|---|---|
| **条件付き確率** | 事象Bが起きた条件での事象Aの確率 P(A\|B) |
| **ベイズの定理** | P(A\|B) = P(B\|A)×P(A) / P(B) |
| **事前確率** | データを観測する前の確率（先験確率）|
| **事後確率** | データを観測した後の確率（尤度で更新）|

### 基本統計量
| 用語 | 内容 |
|---|---|
| **平均（期待値）** | 外れ値に影響を受けやすい |
| **中央値（メジアン）** | 外れ値に強い |
| **最頻値（モード）** | 最も出現頻度が高い値 |
| **分散・標準偏差** | ばらつきを表す |
| **共分散** | 2変数の連動性 |
| **相関係数** | -1〜1。因果関係とは別（相関≠因果）|

### 確率分布
| 分布名 | 特徴 |
|---|---|
| **ベルヌーイ分布** | 結果が2値（成功/失敗）の1回試行 |
| **二項分布** | n回試行中の成功回数の分布 |
| **ポアソン分布** | 一定時間・空間内の稀な事象の発生回数 |
| **正規分布（ガウス分布）** | 平均μ・標準偏差σの釣鐘型。±1σ:68%、±2σ:95%、±3σ:99.7% |

### 情報理論
| 用語 | 内容 |
|---|---|
| **エントロピー** | 確率分布の不確実性・純度の低さの指標 |
| **KLダイバージェンス** | 2つの確率分布の非対称的な差異を測定 |
| **JSダイバージェンス** | KLダイバージェンスを対称化・安定化した指標 |

⭐ `固有値・固有ベクトル` `ベイズの定理` `事前確率・事後確率` `正規分布（68-95-99.7則）` `ポアソン分布` `相関≠因果` `エントロピー` `KLダイバージェンス`

## 04 数理統計②

### 統計的推定・検定
| 用語 | 内容 |
|---|---|
| **最尤推定（MLE）** | 観測データが最も生じやすくなるパラメータを推定 |
| **MAP推定** | 事前分布を考慮した最尤推定 |
| **帰無仮説 (H₀)** | 否定したい仮説（効果なし・差がない）|
| **対立仮説 (H₁)** | 採択したい仮説（効果あり）|
| **p値** | 帰無仮説が正しい場合に観測値以上の結果が出る確率 |
| **有意水準** | p値の閾値（通常5%や1%）|
| **第1種の誤り** | 帰無仮説が正しいのに棄却する（偽陽性 / Type I Error）|
| **第2種の誤り** | 帰無仮説が誤りなのに採択する（偽陰性 / Type II Error）|
| **多重検定問題** | 検定を繰り返すことで偽陽性率が増加する問題 |

### 最適化手法
| 用語 | 内容 |
|---|---|
| **勾配降下法** | 損失関数の勾配方向にパラメータを更新 |
| **学習率（η）** | 1回の更新で移動する量。大きすぎると発散、小さすぎると遅い |
| **SGD** | ミニバッチを使った効率的な勾配降下法 |
| **局所最適解** | 損失関数の局所的な最小値 |
| **大域最適解** | 損失関数全体の最小値 |
| **鞍点（サドルポイント）** | 勾配がゼロだが最小でも最大でもない点 |
| **Momentum** | 過去の勾配を蓄積して慣性的に更新 |
| **AdaGrad** | パラメータごとに学習率を調整 |
| **RMSprop** | AdaGradの学習率減衰を改善 |
| **Adam** | Momentum＋RMSpropの組み合わせ。現在最も広く利用 |

### ベイズ推定
| 用語 | 内容 |
|---|---|
| **ベイズ推定** | 事前分布を観測データで更新して事後分布を求める |
| **共役事前分布** | 事後分布が事前分布と同じ分布族になる都合の良い事前分布 |
| **MCMC（マルコフ連鎖モンテカルロ法）** | 事後分布からサンプリングする手法 |
| **DAG（有向非循環グラフ）** | 循環のない有向グラフ。ベイズネットワーク等に使用 |

⭐ `最尤推定` `MAP推定` `帰無仮説` `p値` `有意水準` `第1・2種の誤り` `多重検定問題` `勾配降下法` `学習率` `局所・大域最適解` `鞍点` `SGD` `Adam` `ベイズ推定` `MCMC` `DAG`

## 05 機械学習

### 学習の種類
| 種類 | 特徴 |
|---|---|
| **教師あり学習** | 正解ラベル付きデータで学習。分類・回帰問題に対応 |
| **教師なし学習** | 正解ラベルなし。クラスタリング・次元削減に使用 |
| **半教師あり学習** | 少量のラベル付き＋大量のラベルなしデータを活用 |
| **強化学習** | エージェントが報酬を最大化するよう試行錯誤 |

### 教師あり学習：主要アルゴリズム
| アルゴリズム | 用途 | キーポイント |
|---|---|---|
| **線形回帰** | 回帰 | 最小二乗法。多重共線性に注意 |
| **ロジスティック回帰** | 分類 | シグモイド（2値）/ ソフトマックス（多クラス）|
| **決定木** | 分類・回帰 | ツリー構造、解釈性高、過学習しやすい |
| **ランダムフォレスト** | 分類・回帰 | アンサンブル（バギング）、過学習に強い |
| **SVM** | 分類 | マージン最大化、カーネル法で非線形対応 |
| **k近傍法（KNN）** | 分類 | 近傍k点の多数決。学習フェーズなし（Lazy Learner）|
| **ナイーブベイズ** | 分類 | 特徴独立を仮定したベイズ分類器 |

### アンサンブル学習
| 手法 | 特徴 |
|---|---|
| **バギング** | 並列学習・多数決。ランダムフォレストが代表例 |
| **ブースティング** | 直列学習・誤り修正。XGBoost・AdaBoostが代表例 |

### 教師なし学習
| アルゴリズム | 特徴 |
|---|---|
| **k-means法** | k個のクラスタに分類（非階層型）|
| **階層的クラスタリング** | デンドログラムで可視化。ウォード法で分散を最小化 |
| **PCA（主成分分析）** | 分散が最大となる方向に次元を削減 |

### 評価指標
| 指標 | 計算式 | 重視する場面 |
|---|---|---|
| **正解率（Accuracy）** | (TP+TN)/全数 | バランスの取れたデータ |
| **適合率（Precision）** | TP/(TP+FP) | FP（偽陽性）を減らしたい |
| **再現率（Recall）** | TP/(TP+FN) | FN（偽陰性）を減らしたい（医療診断等）|
| **F値（F1-score）** | 適合率と再現率の調和平均 | バランスよく評価 |
| **AUC** | ROC曲線の下面積（0〜1） | 閾値によらないモデル比較 |
| **MSE** | 誤差の二乗平均 | 回帰。外れ値に大きく影響 |
| **RMSE** | MSEの平方根 | 単位が元データと同じ |
| **MAE** | 誤差の絶対値平均 | 外れ値に強い |
| **決定係数（R²）** | 0〜1。1が理想 | 回帰モデルの当てはまり |

### 過学習と対策
| 概念 | 内容 |
|---|---|
| **過学習** | 訓練データへの過剰適応。汎化誤差が増大 |
| **バイアス** | モデルが単純すぎる誤差（未学習）|
| **バリアンス** | モデルが複雑すぎる誤差（過学習）|
| **L1正則化（Lasso）** | 重みをスパースにする（特徴選択効果）|
| **L2正則化（Ridge）** | 重みを小さく保つ（滑らか）|
| **データ拡張** | 加工で学習データを増やし、過学習を防ぐ |

### モデル選択
| 指標 | 特徴 |
|---|---|
| **AIC** | 複雑さと当てはまりのバランス（小さいほど良い）|
| **BIC** | AICよりデータ数によるペナルティが大きい |
| **オッカムの剃刀** | 精度が同じなら単純なモデルを選ぶ原則 |

⭐ `教師あり・なし・強化学習` `SVM（マージン最大化）` `k近傍法（Lazy Learner）` `ランダムフォレスト（バギング）` `XGBoost（ブースティング）` `k-means` `PCA` `混同行列` `TP/FP/FN/TN` `適合率・再現率トレードオフ` `F値` `AUC` `MSE・RMSE・MAE` `L1（Lasso）・L2（Ridge）正則化` `AIC` `BIC` `オッカムの剃刀`

## 06 DL概要

### ニューラルネットワークの基本
| 用語 | 内容 |
|---|---|
| **単純パーセプトロン** | 入力・出力層のみ。線形分離可能な問題にのみ対応 |
| **多層パーセプトロン（MLP）** | 隠れ層を追加して非線形問題に対応 |
| **重みとバイアス** | ニューラルネットワークが学習するパラメータ |

### 活性化関数
| 関数 | 特徴 | 用途 |
|---|---|---|
| **シグモイド** | 0〜1に圧縮。勾配消失を起こしやすい | 2クラス出力 |
| **tanh** | -1〜1に圧縮。シグモイドより良いが勾配消失あり | 中間層 |
| **ReLU** | 0以下は0、正値はそのまま。現在最も一般的 | 中間層 |
| **Leaky ReLU** | 負値に微小な傾きを与えてニューロンの死滅を防ぐ | 中間層 |
| **ソフトマックス** | 多クラス確率に変換（合計=1）| 多クラス出力 |

### 誤差逆伝播法（バックプロパゲーション）
- 損失関数の勾配を出力層から入力層に向けて計算
- **チェーンルール（連鎖律）** を利用
- 勾配を用いた学習: `w ← w - η × ∂L/∂w`

### 勾配消失・爆発問題
| 問題 | 原因 | 対策 |
|---|---|---|
| **勾配消失** | 活性化関数の微分が小さい（シグモイド等）| ReLU使用、残差接続（ResNet）|
| **勾配爆発** | 勾配が層を通るごとに指数的に大きくなる | 勾配クリッピング |

### バッチ処理と学習
| 用語 | 内容 |
|---|---|
| **バッチ学習** | 全データを一度に使って勾配計算 |
| **ミニバッチ学習** | 一部のデータで勾配計算。一般的手法 |
| **オンライン学習** | データが来るたびに1件ずつ更新 |
| **エポック** | 全訓練データを1周学習すること |

### 正規化・正則化テクニック
| 技法 | 効果 |
|---|---|
| **バッチ正規化（BatchNorm）** | 各バッチの活性化値を正規化。学習安定化・高速化 |
| **ドロップアウト** | 学習時にランダムにニューロンを無効化。過学習防止 |
| **早期停止（Early Stopping）** | 検証誤差が増大し始めた時点で学習を停止 |

### CNN・RNN
| 用語 | 内容 |
|---|---|
| **畳み込み層** | フィルタ（カーネル）を適用してパターンを抽出 |
| **プーリング層** | 特徴マップのサイズを縮小（最大プーリング等）|
| **パディング** | 入力の周囲にゼロを追加してサイズを維持 |
| **ストライド** | フィルタを何ピクセルずらして適用するか |
| **RNN** | 系列データ処理。前の状態（隠れ状態）を次の入力に使用 |
| **LSTM** | 長・短期記憶を分離したゲート機構で長期依存を解決 |
| **GRU** | LSTMを簡略化した高効率版 |

⭐ `単純パーセプトロン` `多層パーセプトロン（MLP）` `活性化関数` `ReLU（現在主流）` `ソフトマックス` `誤差逆伝播法・チェーンルール` `勾配消失（シグモイド問題）` `勾配クリッピング` `ミニバッチ学習` `エポック` `バッチ正規化` `ドロップアウト` `早期停止` `CNN畳み込み・プーリング` `RNN` `LSTM` `GRU`

## 07 DL要素技術

### Attention機構とTransformer
| 用語 | 内容 |
|---|---|
| **Attention機構** | 系列の各要素が他の要素を参照する重み付き注意機構 |
| **Self-Attention** | 系列内の各要素が互いに関係性を計算 |
| **Multi-Head Attention** | 複数の注意ヘッドで異なる表現を同時に学習 |
| **Transformer** | Attentionのみに基づいた並列処理可能なモデル（2017, Google「Attention is All You Need」）|
| **位置エンコーディング** | Transformerに系列の位置情報を与えるための工夫 |
| **エンコーダ / デコーダ** | 入力を変換 / 出力を生成する部分 |

### BERTとGPT系モデル
| モデル | 特徴 |
|---|---|
