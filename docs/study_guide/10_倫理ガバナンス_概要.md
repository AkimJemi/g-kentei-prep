# 10 倫理ガバナンス ── 概要・重要用語集

> **G検定 クイックリファレンス** | 分野：倫理ガバナンス | 2026-02-25

---

## 📖 章の概要

AIの倫理・社会的課題（バイアス・プライバシー・説明責任）、国内外のガイドライン、AIガバナンスの仕組み、クライシスマネジメントを扱う章。

---

## 🗂️ 大項目一覧と要点

### 法と倫理の基礎
| 用語 | 内容 |
|---|---|
| **ハードロー** | 法律等の法的強制力を持つルール |
| **ソフトロー** | ガイドライン・自主規制等の拘束力の弱いルール |
| **AIガバナンス** | AIの開発・運用を規則に基づいて管理・統治する仕組み |
| **リスクベースアプローチ** | AIのリスクレベルに応じて規制の強度を変える考え方 |

### 主要AIガイドライン
| 国・機関 | ガイドライン | ポイント |
|---|---|---|
| **日本（内閣府）** | 人間中心のAI社会原則 | 7原則（人間中心・教育・プライバシー・セキュリティ・公正競争・公平性・イノベーション） |
| **EU** | EU AI Act（AI法） | リスクレベル（禁止/高/限定/最小）で規制を段階化。世界初の包括的AI法規制 |
| **EU** | 信頼できるAIのための倫理ガイドライン | 適法性・倫理性・堅牢性の3要素、7つの要件 |
| **OECD** | OECD AI原則 | 5原則（包摂的成長・透明性・説明責任等）。G20にも採択 |
| **UNESCO** | AI倫理に関する勧告 | 文化の多様性・環境・デジタル格差を重視 |
| **米国** | 大統領令・AI Bill of Rights | AI安全性・市民権保護を推進 |

### AIの公平性（FAT）
| 用語 | 内容 |
|---|---|
| **Fairness（公平性）** | 特定のグループへの不当な差別がないこと |
| **Accountability（説明責任）** | AI判断プロセスを説明できること |
| **Transparency（透明性）** | 動作原理・データ利用を可能な限り開示 |
| **サンプリングバイアス** | 抽出データ自体に偏りがある状態 |
| **アルゴリズムバイアス** | 偏ったデータを学習してモデルが偏った判断をする |
| **センシティブ属性** | 性別・国籍・宗教等の繊細な扱いが必要な属性 |
| **代理変数** | センシティブ属性と相関が高く、間接的に推定できる変数 |

### プライバシー
| 用語 | 内容 |
|---|---|
| **プライバシー・バイ・デザイン（PbD）** | 設計段階からプライバシー保護を組み込む考え方 |
| **カメラ画像利活用ガイドブック** | 経産省・総務省が策定したカメラ利用の指針 |

### AIガバナンスの要素
| 要素 | 内容 |
|---|---|
| **AIポリシー** | 企業のAI開発・利用に関する具体的な指針 |
| **倫理アセスメント** | AI開発が倫理的に適切か評価するプロセス |
| **責任の所在** | 誰がどの責任を負うかを明確化 |
| **人間の関与** | 重要判断は人間が最終決定 |
| **再現性** | 同じ入力で安定した出力が得られること |
| **トレーサビリティ** | AI開発プロセスを追跡できること |
| **文書化** | 開発プロセスや決定事項を記録に残す |

### クライシスマネジメント
| 用語 | 内容 |
|---|---|
| **クライシス・マネジメント** | 危機が発生した後の対応・被害最小化 |
| **リスク・マネジメント** | 危機を未然に防ぐための予防策 |
| **コーポレートガバナンス** | 企業経営の透明性・健全性を監視する仕組み |
| **内部統制** | 社内のルール・仕組みで不祥事を防ぐ |
| **ELSI** | Ethical, Legal and Social Issues（倫理的・法的・社会的問題） |
| **バリュー・センシティブ・デザイン** | 設計段階から価値全般（プライバシー等）に配慮 |
| **シリアス・ゲーム** | 社会問題をゲームで体験・学習するツール |

### AI特有の倫理的課題
| 課題 | 内容 |
|---|---|
| **自律型兵器（AWS）** | 人間の介在なしに攻撃対象を決定・実行する兵器 |
| **LAWS（自律型致死兵器システム）** | AWSのうち特に殺傷能力を持つもの |
| **死者への敬意** | 故人の音声・映像をAIで生成することの倫理問題 |
| **人間の自律性** | AI情報提供によって人間の判断が操作される懸念 |
| **フェイクニュース** | AI生成の誤情報・偽情報の拡散 |
| **Tayの事例** | Microsoftの会話ボットが差別的発言を学習した事例 |
