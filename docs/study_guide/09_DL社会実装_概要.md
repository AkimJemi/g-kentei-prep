# 09 DL社会実装 ── 概要・重要用語集

> **G検定 クイックリファレンス** | 分野：DL社会実装 | 2026-02-25

---

## 📖 章の概要

AIを実際のビジネスやシステムに組み込む際の実践的知識。データ収集・前処理、システム設計、MLOps、プロジェクト管理、各産業への応用を扱う章。

---

## 🗂️ 大項目一覧と要点

### AIプロジェクトの進め方
| ステップ | 内容 |
|---|---|
| **問題定義** | 解くべき問題を明確化。KPIを設定 |
| **データ収集・確認** | 量・質・偏りを確認。必要なラベル付け計画 |
| **データ前処理** | クレンジング・正規化・特徴量エンジニアリング |
| **モデル選択・学習** | 適切なアルゴリズムを選択して学習 |
| **評価・チューニング** | 検証データで評価、ハイパーパラメータ調整 |
| **デプロイ** | 本番環境への展開 |
| **モニタリング** | 精度・ドリフト・異常を継続的に監視 |

### データ前処理
| 処理 | 内容 |
|---|---|
| **欠損値処理** | 削除・平均値補完・多重代入法など |
| **外れ値処理** | 除去またはキャップリング（上限設定） |
| **正規化** | [0,1]に変換（Min-Max正規化） |
| **標準化** | 平均0・分散1に変換（Zスコア正規化） |
| **ラベルエンコーディング** | カテゴリ変数を数値に変換 |
| **One-Hot Encoding** | カテゴリ変数をバイナリベクトルに変換 |
| **データ拡張** | 既存データを加工して水増し（画像回転・反転等） |

### MLOps（機械学習の運用）
| 用語 | 内容 |
|---|---|
| **MLOps** | ML開発と運用を継続的に改善するプラクティス |
| **CI/CD （継続的統合・デプロイ）** | モデルの自動テスト・デプロイパイプライン |
| **データドリフト** | 本番データの分布が学習データから乖離する問題 |
| **モデルドリフト** | 時間とともにモデルの精度が低下する現象 |
| **A/Bテスト** | 2つのモデルをユーザーに向けて同時に検証 |
| **シャドウデプロイ** | 本番トラフィックで並列実行して比較 |
| **フィーチャーストア** | 特徴量を一元管理・再利用するシステム |
| **モデルレジストリ** | 学習済みモデルを管理・バージョン管理するシステム |

### ハードウェアとインフラ
| 用語 | 内容 |
|---|---|
| **GPU** | 並列計算に特化。ディープラーニング学習に不可欠 |
| **TPU（Tensor Processing Unit）** | Googleが開発したAI専用チップ |
| **FPGA** | 現場でプログラム可能な集積回路。エッジ向き |
| **分散学習** | 複数GPUや複数マシンで並列学習（データ並列・モデル並列） |
| **コンテナ（Docker）** | 環境の再現性・移植性を確保 |
| **Kubernetes** | コンテナのオーケストレーション（管理・スケーリング） |

### 主要産業への応用
| 分野 | 代表的なAI活用 |
|---|---|
| **製造業** | 不良品検出、予知保全、ロボットビジョン |
| **医療・ヘルスケア** | 画像診断（がん検出等）、創薬支援、電子カルテ分析 |
| **金融** | 不正検知、信用スコアリング、アルゴリズムトレーディング |
| **農業（スマート農業）** | ドローン監視、農作物病害虫検出、収量予測 |
| **交通・自動運転** | 物体検出、経路計画、行動予測 |
| **エネルギー** | 需要予測、設備異常検知、再生可能エネルギー最適化 |
| **小売・EC** | レコメンデーション、需要予測、在庫最適化 |

### 自動運転レベル（SAEレベル）
| レベル | 内容 |
|---|---|
| **Lv.0** | 非自動化（ドライバーが全て操作） |
| **Lv.1** | 運転支援（ACC・レーンキープ等の単機能） |
| **Lv.2** | 部分自動化（複数機能を同時制御） |
| **Lv.3** | 条件付き自動化（限定条件下で自動。ドライバーは必要） |
| **Lv.4** | 高度自動化（限定条件下で完全自動） |
| **Lv.5** | 完全自動化（あらゆる条件で無人運転可能） |

### 評価指標の実務
| 用語 | 内容 |
|---|---|
| **KPI（Key Performance Indicator）** | プロジェクト成功を測る指標 |
| **PoC（Proof of Concept）** | 実現可能性の検証実験 |
| **ROI（投資対効果）** | AI導入コストに対して得られた利益の割合 |
