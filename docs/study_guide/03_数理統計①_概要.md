# 03 数理統計① ── 概要・重要用語集

## 📖 章の概要

機械学習に不可欠な数学・統計の基礎。線形代数（行列・ベクトル）、確率論、基本統計量、確率分布を扱う章。

---

## 🗂️ 大項目一覧と要点

### 線形代数の基礎
| 用語 | 内容 |
|---|---|
| **スカラー** | 大きさのみを持つ数値 |
| **ベクトル** | 大きさ＋向きを持つ量。n次元ベクトル |
| **行列** | 数値を矩形状に並べたもの（m×n行列） |
| **転置行列** | 行と列を入れ替えた行列（$A^T$） |
| **内積** | ベクトル同士の積のスカラー。類似度計算に使用 |
| **行列の積** | (m×k) × (k×n) = (m×n) 行列 |
| **単位行列** | 対角成分が1、他が0の正方行列（$I$） |
| **逆行列** | $A \cdot A^{-1} = I$ となる行列 |
| **固有値・固有ベクトル** | $Ax = \lambda x$ を満たす $\lambda$（固有値）と $x$（固有ベクトル） |

### 確率の基礎
| 用語 | 内容 |
|---|---|
| **条件付き確率** | 事象Bが起きた条件での事象Aの確率 $P(A\|B)$ |
| **ベイズの定理** | $P(A\|B) = \frac{P(B\|A)P(A)}{P(B)}$ |
| **事前確率** | データを観測する前の確率（先験確率） |
| **事後確率** | データを観測した後の確率（尤度で更新） |
| **独立** | $P(A \cap B) = P(A) \cdot P(B)$ |

### 基本統計量
| 用語 | 内容 |
|---|---|
| **平均（期待値）** | データの重心。外れ値に影響を受けやすい |
| **中央値（メジアン）** | 中央の値。外れ値に強い |
| **最頻値（モード）** | 最も出現頻度が高い値 |
| **分散** | データのばらつき。偏差の二乗平均 |
| **標準偏差** | 分散の平方根。単位が元データと同じ |
| **共分散** | 2変数の連動性を示す量 |
| **相関係数** | -1〜1 の範囲。1に近いほど正の相関 |

### 確率分布
| 分布名 | 特徴・用途 |
|---|---|
| **ベルヌーイ分布** | 結果が2値（成功/失敗）の1回試行 |
| **二項分布** | n回試行中の成功回数の分布 |
| **ポアソン分布** | 一定時間・空間内の稀な事象の発生回数 |
| **正規分布（ガウス分布）** | 平均μ・標準偏差σの釣鐘型。多くの自然現象を表す |
| **標準正規分布** | 平均0・標準偏差1に正規化した分布 |
| **一様分布** | 指定範囲内で等確率に分布 |

### 情報理論
| 用語 | 内容 |
|---|---|
| **エントロピー** | 確率分布の不確実性。純度の低さの指標 |
| **KLダイバージェンス** | 2つの確率分布の非対称的な差異を測定 |
| **JSダイバージェンス** | KLダイバージェンスを対称化・安定化した指標 |
